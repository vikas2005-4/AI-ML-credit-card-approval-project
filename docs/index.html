Project architecture

How the application is structured

  • Frontend: HTML, CSS, JS for landing pages and form interactions.
  • Backend: Flask handles routing, validation, and templates.
  • Machine Learning: XGBoost and scikit-learn power prediction logic.
  • Data: SQLite stores prediction history and user activity.
  • Deploy: Render hosts the Flask app with Gunicorn.
