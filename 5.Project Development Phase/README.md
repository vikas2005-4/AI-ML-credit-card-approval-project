# Credit Card Approval Prediction

This project is a beginner-friendly end-to-end credit approval prediction system built with Flask and ML libraries.

Structure: follow the folders inside this directory.

To run locally:

1. Install dependencies: `pip install -r requirements.txt`
2. (Optional) Place your dataset at `dataset/credit_card.csv` or the app will try to locate one in the repository root.
3. Train models: `python src/train_model.py`
4. Run the app: `python app.py`
